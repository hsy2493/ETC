# <인터넷 강의 사이트(3회차 프로젝트)>
1. 작업 기간 : 2024. 05. 27 ~ 2024. 06. 17<br>
2. 주제 : 온라인 인터넷 강의 사이트
3. 목적 : 수험생을 대상으로 고객이 온라인에서 ETC 사이트를 통하여 희망하는 해당 과목의 월 구독권을 구매함으로써, 인터넷 강의 동영상과 교재로 학습하는 것을 주목적으로 진행된 인터넷 강의 사이트 프로젝트입니다.<br>
4. 주요 기능 : 로그인, 회원가입<br>
